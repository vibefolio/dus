## 1. 현재 상태 진단

### 개발 완성도
- **코드 규모**: 12개 모델, 15개 DB 테이블, Rails 8.1 풀스택
- **아키텍처**: 멀티테넌시 (서브도메인 기반), Hotwire (Turbo+Stimulus)
- **주요 기능**: 디자인 템플릿 마켓플레이스, 견적 시스템, 에이전시 관리, OAuth 인증
- **프론트엔드**: Tailwind CSS 커스텀 컴포넌트 시스템 (glassmorphism, 그린 컬러 정체성)
- **배포**: NCP Docker (Fly.io에서 이관 완료, 2026-03-24)

### 매출/사용자 현황 (추정)
- **상태**: 초기 단계 (출시 준비 완료, 실배포 전)
- **도메인**: designd.co.kr (메인), dus.vibers.co.kr (서브)
- **타겟 고객**: 웹 에이전시, 개인 디자이너
- **비즈니스 모델**: B2B 템플릿 마켓플레이스 + 에이전시 플랫폼

### 핵심 과제
1. **PMF(Product-Market Fit) 미검증**: 실제 에이전시의 니즈와 템플릿의 적합성 미확인
2. **멀티테넌시 복잡성**: 에이전시 계층 구조(재귀)가 복잡, 데이터 격리 보장 필요
3. **경쟁 차별화**: 템플릿 마켓플레이스의 차별성 (Figma, Canva와의 차이점 불명확)
4. **초기 사용자 확보**: 콜드 스타트 문제 (초기 템플릿 콘텐츠 부족)
5. **운영 기술 채무**: Google OAuth 통합 완료되었으나, Devise 보안 강화 필요

## 3. Beachhead 분석

### Primary Segment: 1-3인 한국 웹 에이전시

**4가지 선정 기준:**

1. **Segment Size**: 한국 웹 에이전시 약 2,000-3,000개 (통계청 기준)
2. **Execution Advantage**:
   - Vibers 자체가 웹 에이전시 (자체 고객으로 시작 가능)
   - 한국 시장 이해도 높음
   - Figma/Canva 대비 한국 특화 기능(한글, 로컬 결제, 견적서)
3. **Size of Beachhead**: 초기 월 50-100개 에이전시 확보 가능
4. **Growth Path**: 에이전시 → 개인 디자이너(1차 확장) → 기업 마케팅팀(2차 확장)

### Go-to-Market 전략

- **Phase 1 (0-3개월)**: Vibers 자체 + 네트워크 내 에이전시 10개사 베타 테스트
- **Phase 2 (3-6개월)**: 에이전시 커뮤니티(온라인 포럼, 오프라인 네트워킹) 진출
- **Phase 3 (6-12개월)**: 크리에이터 마켓플레이스 활성화 (템플릿 공급 다양화)

## 4. Growth Loop

### Best-fit Loop: **Platform Network Effect (크리에이터↔에이전시)**

```
크리에이터 (공급) → 고품질 템플릿 축적
                ↓
에이전시 (수요) → 템플릿 구매 & 사용 증가
                ↓
크리에이터 수익 증대 → 더 많은 크리에이터 유입
```

**Loop 강도**:
- 루프 시간: 4-6주 (월간 신규 템플릿 20개 목표)
- 확대 점: 템플릿 품질 + 판매 수수료 투명성
- 제약: 초기 크리에이터 모집 어려움 (닭과 계란 문제)

### 보완 Loop: **Viral Referral (에이전시 추천)**

- 기존 고객이 동료 에이전시에 추천 → 마켓플레이스 인지도 증가
- 인센티브: 추천당 수수료 1% 보너스

## 5. 가격 전략

### 현재 상태
- 템플릿: 정해진 가격 미상 (설정 필요)
- 주문 수수료: 구현되지 않음
- 에이전시 구독: 미계획

### 권장 가격 모델 (한국 시장 기준)

#### 템플릿 (Marketplace)
| 등급 | 가격 | 대상 | 예시 |
|------|------|------|------|
| 무료 | 0원 | 초보자, 샘플 | 기본 포트폴리오 (1개) |
| 기본 | 5,000-10,000원 | 개인 디자이너 | 반응형 웹사이트 (3개) |
| 프리미엄 | 15,000-30,000원 | 중소 에이전시 | 고급 템플릿 (5개) + 피그마 소스 |
| 번들 | 50,000-100,000원 | 에이전시 | 월간 신규 템플릿 무제한 구독 |

#### 에이전시 구독 (선택사항)
| 플랜 | 월가격 | 기능 | 리소스 |
|------|--------|------|--------|
| Free | 0원 | 템플릿 쇼핑, 3개 에이전시 계정 | 1GB 저장소 |
| Pro | 29,000원 | 5개 에이전시, 주문 관리, 견적서 자동화 | 10GB 저장소 |
| Business | 99,000원 | 무제한 에이전시, API 접근, 커스텀 도메인 | 100GB 저장소 |

#### 주문 수수료 (Commission)
- 기본 수수료: 주문 금액의 3% (결제 게이트웨이 비용 포함)
- 월 거래 건수 100건 이상: 2.5%로 인하

### 가격 전략 근거
- 한국 중소 에이전시 월 예산: 템플릿 5-10만원 범위 (설문 필요)
- Figma: $12/월 (팀), Canva Pro: $120/년 → DUS는 이보다 저가 포지셔닝
- 수익 구조: 템플릿 마진(70%) + 구독료(80% gross margin) + 수수료(3%)

## 7. North Star Metric

### 추천 NSM
**"월간 활성 에이전시(MAA)당 템플릿 판매액"**
- 초기 목표: $50 (12개월 후)
- KPI: (월 총 템플릿 판매액) / (월간 활성 에이전시)

### Input Metrics (주간 추적)
1. **Supply Side**: 활성 크리에이터 수, 신규 템플릿 발행 수, 크리에이터 만족도 (NPS)
2. **Demand Side**: 신규 에이전시 가입 수, 활성 사용자 수, 세션당 템플릿 조회 수
3. **Conversion**: 템플릿 클릭→구매 전환율, 평균 주문가액 (AOV)
4. **Retention**: 에이전시 30일 재방문율, 크리에이터 월간 활성율
