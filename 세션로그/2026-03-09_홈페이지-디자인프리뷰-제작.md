# 홈페이지 디자인 프리뷰 제작

**날짜:** 2026년 3월 9일 (월)
**세션 유형:** 디자인 프리뷰 제작

---

## 결정 사항

- **업종:** 디자인+개발 통합 서비스
- **분위기:** 밝고 친근한
- **섹션 구성:** 히어로 + 서비스 + 포트폴리오 + 프로세스 + CTA + 푸터

## 제작 내용

### 파일: `디자인프리뷰_홈페이지.html`

Tailwind CSS CDN + Pretendard 폰트 기반으로 제작한 풀 페이지 디자인 프리뷰.

### 포함된 섹션

1. **네비게이션** - 고정형, 블러 배경, 로고 + 메뉴 + CTA 버튼
2. **히어로** - 그라디언트 배경, 카피, 신뢰 지표(150+ 프로젝트, 98% 만족도)
3. **서비스** - 4개 카드(UI/UX, 웹 개발, 앱 개발, 브랜딩), 호버 애니메이션
4. **포트폴리오** - 필터 탭 + 3개 카드 그리드, 오버레이 호버 효과
5. **프로세스** - 4단계(상담→디자인→개발→런칭)
6. **CTA** - 블루-퍼플 그라디언트 배경, 견적 문의/이메일 버튼
7. **푸터** - 4열 레이아웃(회사정보, 서비스, 회사, 연락처)

### 디자인 특징
- 컬러: 블루(#3b82f6) + 퍼플(#d946ef) 그라디언트 포인트
- 폰트: Pretendard
- 카드 호버 시 Y축 이동 + 그림자 효과
- 플로팅 애니메이션 배경 장식
- 반응형 대응 (모바일/태블릿/데스크톱)
