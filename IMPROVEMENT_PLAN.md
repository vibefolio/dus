# DLAB 웹사이트 개선 계획 (Top 10)

## 1. 퍼포먼스 및 UX

1. **이미지 최적화 (Priority: High)**

   - 현황: Unsplash 고해상도 이미지 직접 로드 중
   - 개선: Cloudinary 또는 ActiveStorage Variants 도입하여 WebP 포맷 및 사이즈 최적화 제공
   - 효과: LCP(가장 큰 콘텐츠 페인트) 시간 50% 이상 단축

2. **페이지 전환 애니메이션 (Priority: Medium)**

   - 현황: 기본 브라우저 페이지 로드
   - 개선: Turbo Frame + CSS Transition 활용하여 부드러운 페이드/슬라이드 효과 적용
   - 효과: 네이티브 앱 같은 프리미엄 사용자 경험 제공

3. **실시간 상담 톡 (Priority: High)**
   - 현황: 이메일 폼 문의만 가능
   - 개선: 채널톡 또는 카카오 상담톡 플러그인 연동
   - 효과: 잠재 고객 이탈 방지 및 문의 전환율 상승

## 2. SEO 및 마케팅

4. **동적 Open Graph 태그 (Priority: High)**

   - 현황: 기본 타이틀만 설정됨
   - 개선: 각 템플릿/포트폴리오 별 썸네일과 설명을 og:image, og:description에 동적 할당
   - 효과: 카카오톡/슬랙 공유 시 클릭률 상승

5. **블로그/인사이트 섹션 (Priority: Medium)**

   - 현황: 정적 콘텐츠 위주
   - 개선: Markdown 기반의 기술 블로그 기능 추가
   - 효과: 롱테일 키워드 검색 유입 증가

6. **뉴스레터 구독 (Priority: Low)**
   - 현황: 구독 기능 없음
   - 개선: 푸터에 구독 폼 배치 및 Mailchimp 연동
   - 효과: 잠재 고객 DB 확보

## 3. 기능 및 안정성

7. **폼 유효성 검사 강화 (Priority: High)**

   - 현황: 서버 사이드 검증 위주
   - 개선: StimulusJS로 실시간 입력 검증 (전화번호 포맷팅, 이메일 확인)
   - 효과: 잘못된 데이터 제출 방지 및 500 에러 감소

8. **관리자 대시보드 시각화 (Priority: Medium)**

   - 현황: 단순 숫자 통계
   - 개선: Chart.js 도입하여 월별 문의량, 인기 템플릿 그래프 표시
   - 효과: 비즈니스 인사이트 도출 용이

9. **다국어 지원 (i18n) (Priority: Low)**

   - 현황: 한국어 전용
   - 개선: Rails i18n 적용으로 영문 페이지 추가
   - 효과: 글로벌 클라이언트 대응 가능

10. **다크 모드 (Priority: Low)**
    - 현황: 라이트 모드 고정
    - 개선: Tailwind 다크 모드 활성화 및 색상 토큰 정의
    - 효과: 사용자 선호도 존중 및 모던한 이미지 강화
