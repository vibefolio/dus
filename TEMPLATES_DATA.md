# 33개 디자인 템플릿 데이터

## 자동 추가 방법 (배포 서버)

배포 서버(Render.com)에서 다음 명령 실행:

```bash
rails db:seed
```

이 명령은 아래 33개 템플릿을 자동으로 DB에 추가합니다.
